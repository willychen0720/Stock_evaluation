# 📊 專業經理人價值管理系統 (Forward 12M 滾動預測版)

一套基於 **滾動 12 個月 (Forward 12M)** 法人邏輯設計的台股量化估值系統。專為克服傳統財務模型「時效滯後性」與「新股/飆股數據污染」而設計。

## 🎯 核心優化亮點
1. **跨年度滾動 12M 預測**：自動感應最新公告季報（Q1/Q2/Q3）與最新月營收進度差，自動調配跨年度歷史基期，永遠領先市場半年反映基本面。
2. **實時時間風控鎖**：自動阻斷未來未公告月份，避免第三方 API 數據超前造成的營收重複計算。
3. **新股無基期安全遞補**：針對如 `7711 永擎` 等歷史基期資料缺漏之新掛牌個股，自動啟用「最新單季平均營收」進行嚴謹防守遞補。
4. **高階 Re-rating 修正機制**：內建 AI 先進封裝、LEO 衛星等結構性紅利修正係數 ($\alpha$) 與級距風控定錨指引。

## ⚙️ 系統架構
* `app.py`：Streamlit 前端控制台與動態看板
* `data_fetcher.py`：FinMind 大數據跨年調度與髒數據洗滌中心
* `model_engine.py`：Forward 12M 核心估值運算引擎
* `batch_update_pe.py`：歷史 3 年 PE 區間大數據滾動計算
